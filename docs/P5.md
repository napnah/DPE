# P5 — Web UI (Dashboard + Group + Editor)

## Scope

| 路由 | 功能 |
|------|------|
| `/` | 首次生成 Ed25519 身份（UID） |
| `/dashboard` | 总面板：网络/邻居、建群、邀请、群列表 |
| `/groups/:groupId` | 群组：文档树、成员、ACL、P2P 信令 |
| `/groups/:groupId/docs/:docId` | Yjs 文档编辑 + SignedUpdate P2P 同步 |

## API 使用

- `control-plane`：群组、邀请、JWT、`/tree`、`/members`、`SetACL` RPC
- `lan-agent`：`/network`、`/discovery`、`/peers?uid=`
- `signaling` + 浏览器 **WebRTC**：群组房间内 mesh，AuthEnvelope 后同步

## 本地运行

```bash
docker compose up -d postgres
cd apps/control-plane && pnpm db:push
pnpm dev   # web :5173, control-plane :3001, signaling :3002, lan-agent :3003
```

打开 http://localhost:5173 ，生成身份后使用总面板。

## Verify

```bash
pnpm verify:p5
```

## Notes

- 编辑器为 **Y.Text + textarea**（MVP）；富文本 TipTap 可作为后续增强。
- 文档状态另存 `localStorage` 作离线草稿；权威合并仍走 P2P SignedUpdate。
- 需先 **建群/入群** 以保存 `pk_admin` 到 localStorage。
