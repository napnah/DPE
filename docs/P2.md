# P2 — Control Plane (Admin)

## Scope

PostgreSQL-backed NestJS service for group lifecycle, invitations, JWT issuance, ACL/RPC, proxy governance, and doc key rotation metadata.

## APIs

| Method | Path | Description |
|--------|------|-------------|
| POST | `/groups` | Create group (default `control_mode: proxy`) |
| GET | `/users/me/groups?node_id=&role=owner\|member` | List groups |
| POST | `/groups/:id/join` | Join with `{ node_id, public_key }` |
| POST | `/groups/:id/invitations?inviter_node_id=` | Create invitation |
| GET | `/users/me/invitations?node_id=` | Pending invitations |
| POST | `/invitations/:id/accept` | Accept (+ root read ACL) |
| POST | `/invitations/:id/reject?node_id=` | Reject |
| POST | `/groups/:id/jwt/refresh` | Issue JWT with sealed `doc_key` |
| GET | `/groups/:id/tree?node_id=` | Visible doc nodes |
| POST | `/groups/:id/rpc?caller_node_id=` | `SetACL`, `CreateChild`, `DeleteDoc` |
| POST | `/groups/:id/docs/:docId/rotate-key?caller_node_id=` | Bump `key_version` |
| POST | `/groups/:id/governance/enable-proxy` | Owner-signed enable proxy |
| POST | `/groups/:id/governance/disable-proxy` | Owner-signed disable proxy |

## Env

- `DATABASE_URL` — PostgreSQL (see `docker compose up postgres`)
- `DPE_SIGNING_PRIVATE_KEY` / `DPE_SIGNING_PUBLIC_KEY` — base64url Ed25519 issuer keys (optional; ephemeral if unset)
- `DPE_PROXY_BASE_URL` — advertised proxy URL

## Dev

```bash
docker compose up -d postgres
cd apps/control-plane
pnpm db:push
pnpm dev
```

## Verify

```bash
pnpm verify:p2          # build + unit tests
pnpm verify:p2 --api    # + Postgres smoke (needs DB)
```

## Notes

- `DocumentKey` stores doc keys server-side for JWT sealing (**dev only**; use HSM/KMS in production).
- Clients pin `issuerPublicKey` (`pk_admin` in API responses) when verifying JWTs.
- Full web UI wiring → **P5**; signaling/P2P auth → **P3**.
